How to use this file for Gmail testing

  • Step 1: Confirm GitHub Pages is enabled for this repo (Settings → Pages → Source: Deploy from branch → Branch: main → Folder: / (root)).
  • Step 2: Wait 1–2 minutes after enabling Pages for assets to go live.
  • Step 3: Verify assets load at: https://petrikin83.github.io/cdc-premium-signature-prototype/public/
  • Step 4: Open this file in Chrome. Select all content inside the signature block (the white card). Copy. Paste into Gmail Settings → Signature.
  • Step 5: Send a test email to yourself. Confirm images load.
