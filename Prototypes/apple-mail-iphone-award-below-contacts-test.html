Prototype only. This file tests the approved Apple Mail / iPhone layout direction with the award banner below contacts. If approved, the live generator template (js/templates/apple-mail-template.js) will be updated separately. The Gmail template and all generator behavior are unchanged.

Design decisions

Outer table width 340px — validated in Phase 2; resolved iPhone Settings clipping; preserves identity breathing room
Logo width 92px — same as approved template; recognizable, not oversized
Identity column 236px (340 − 92 − 12 spacer = 236px)
Award image width 200px — 59% of table width; left-aligned; legible FT text at this size; 140px negative space to the right reads as "credential badge" not hero banner; wider would compete with identity, narrower would look incidental
Contacts structure One <tr> per contact line — maximum per-row bgcolor protection against dark mode inversion
Address placement Below contacts, above award; no icon; 10.5px/lighter color — visually subordinate; conditional in generator (not rendered if empty)
Award placement Full-width row above lower divider; sits naturally as a credential conclusion to the contact block
Dark mode protection bgcolor="#ffffff" on every td; background-color:#ffffff inline on every cell and text container; explicit text colors on all elements; style block with -webkit-text-size-adjust and color-scheme:light

How to test on iPhone

  1. Open this file in Safari on iPhone (or transfer it via AirDrop / iCloud Drive).
  2. Long-press the signature table → Select All → Copy.
  3. Go to Settings → Mail → Signature.
  4. Tap the signature field → select all existing text → Paste.
  5. Verify: no horizontal clipping on the right edge, white background preserved, all text readable.
  6. Switch iPhone to Dark Mode and verify: background remains white, text remains readable, no color inversion on any element.
  7. Compose a new email and check the signature in the compose view and in a received email.

If approved, this layout replaces the current Apple Mail template in js/templates/apple-mail-template.js. The Gmail template is not affected.
